Judul Proyek
AP-2-HEALTH

Deskripsi Singkat
AP-2-Health adalah program berbasis Python yang dirancang untuk membantu pengguna memantau dan meningkatkan gaya hidup sehat melalui berbagai fitur interaktif. 

Tujuan 
Program ini mengajak pengguna untuk lebih sadar terhadap kondisi tubuh serta membangun kebiasaan hidup sehat secara konsisten dan modern.

Fitur Utama
1. Hitung BMI
2. Smartfat Calculator
3. Health Check (durasi tidur, konsumsi air, durasi olahraga)
4. Set Health Mission
5. Health Progress
6. How Are You Today
7. EyeCheck

Penjelasan Fitur
Berikut adalah penjelasan rinci dari setiap fitur:
1. Fitur Hitung BMI memungkikan pengguna untuk mengetahui BMI (Body Mass Index) dan menilai apakah berat badan pengguna tergolong kurus, ideal, berlebih, atau obesitas dan dihitung berdasarkan berat badan (kg) dan tinggi badan (cm).

2. Fitur Smartfat Calculator memungkinkan pengguna untuk menghitung kadar lemak didalam tubuhnya serta memberi tahu pengguna mengenai status/kategori keadaan kadar lemak dalam tubuh.
   
3. Fitur Health Check memberikan saran dan evaluasi kesehatan harian berdasarkan kebiasaan pengguna, seperti pola tidur, asupan air, dan durasi olahraga. Program menyesuaikan rekomendasi dengan usia pengguna, membantu mereka menjaga keseimbangan dan membangun gaya hidup sehat secara konsisten.
   
4. Fitur Set Health Mission merupakan bagian dari sistem AP-2-Health, yaitu aplikasi berbasis Python yang membantu pengguna mengatur dan memantau target gaya hidup sehat harian. Fokus utama dari kode ini untuk mengatur, melihat, dan menghapus target harian pengguna mencakup workout, kalori, dan protein  serta menyimpannya ke dalam database.
 
5. Fitur Health Progress membantu pengguna memantau pencapaian harian seperti olahraga, kalori, dan protein. Data yang diinput akan disimpan dan ditampilkan kembali dalam bentuk riwayat, lengkap dengan status keberhasilan agar pengguna tahu sejauh mana progres kesehatannya. Dengan fitur ini, pengguna dapat mengevaluasi kebiasaan sehatnya setiap hari dan termotivasi untuk terus mencapai target kesehatan pribadi.
   
6. Fitur How are you today untuk mengevaluasi suasana hati pengguna melalui lima pertanyaan. Setiap pertanyaan dijawab dengan skala 1–5. Program memvalidasi input, menghitung rata-rata dari seluruh jawaban, lalu menampilkan hasil evaluasi mood beserta pesan yang sesuai dengan kategori nilai rata-ratanya.
   
7. Fitur EyeCheck berfungsi untuk membantu mengidentifikasi tingkat keparahan Computer Vision Syndrome (CVS) akibat penggunaan layar secara berlebihan. Melalui pertanyaan tentang frekuensi dan intensitas gejala, program menghitung skor untuk menentukan kategori kondisi mata dari tidak ada gejala hingga CVS berat serta memberikan saran praktis untuk menjaga kesehatan mata.
   
Cara Menjalankan Program
Untuk menjalankan proyek AP-2-Health, ikuti langkah-langkah berikut:
1. Pastikan sudah menginstal Python dan Git di perangkat.
2. Clone repository dengan perintah: git clone https://github.com/arenjiee/AP-2-HEALTH.git
3. Masuk ke folder project: cd AP-2-Health
4. Jalankan program utama dengan perintah: python app.py
5. Setelah itu, program akan menampilkan menu utama di terminal dan siap digunakan.

Teknologi yang Digunakan
1. Python
2. SQLite3 berfungsi untuk menyimpan data di database
